Writing a Technical Instruction for a Cafe-Quality Matcha Latte

Project Description

This project involved developing a professional technical instruction document designed to guide beginners in preparing a café-quality matcha latte at home. The instructions emphasize clarity, organization, safety, and precision while explaining traditional Japanese whisking techniques used to produce smooth microfoam and balanced flavor.

The document includes a disclaimer, introduction, required tools, step-by-step procedures, cautions, and cleaning instructions to ensure that readers with no prior experience can successfully follow the process and achieve consistent results.

Skills Learned

  • Technical and professional writing
  • Instructional design
  • Using precise language and safety cautions
  • Process documentation
